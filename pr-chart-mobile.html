PR Menu Bar

See merged pull requests without leaving your desktop. A tiny macOS menu bar app for watching team and personal PR activity across repositories.

1,329 Merged PRs in sample data

Small surface, useful signal.

PR Menu Bar keeps the activity view close to the work without becoming another dashboard to open, tune, or maintain.

GitHub-backed refresh

Fetch merged pull requests across repositories and keep the last useful activity visible between refreshes.

Repository breakdown

Scan the weekly shape of work, then drop into the repositories carrying most of the movement.

Failure-aware updates

Refresh errors do not wipe the view. Rate-limit and token issues are surfaced without hiding previous data.

Prototype-friendly, honest about data.

Use sample mode to explore the app immediately. Add a GitHub token in the launch environment when you want live repository activity.

  • Live data when you provide a token
  • Sample mode when you do not
  • Local checks through the existing Makefile
# run with live GitHub data
$ PR_MENU_BAR_GITHUB_TOKEN=github_pat_xxx make run

# run local checks
$ make ci-local
