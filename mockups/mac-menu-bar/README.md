# PRBar Mac Menu Bar Prototype

Interactive HTML/CSS/JS mockup for the Mac menu bar share and release-card direction.

Open `index.html` directly, or run a local server:

```bash
python3 -m http.server 5174 --directory mockups/mac-menu-bar
```

Then open `http://localhost:5174`.

## Covered Interactions

- Activity / Releases / Settings tabs
- Day / Week / Month activity ranges
- `Share PR Card` fixed preview and export sheet
- release selection and release notes preview
- `Share Release Card` fixed preview and export sheet
- simulated Share, Copy Image, Save PNG, and Copy Notes actions
- repository inclusion toggles that affect Activity and Releases
